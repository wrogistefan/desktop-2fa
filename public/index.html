desktop‑2fa

A secure, offline, local‑first TOTP manager for developers, engineers, and privacy‑focused users.

Why desktop‑2fa?

  • No cloud, no phone, no telemetry
  • Encrypted vault (AES‑GCM + Argon2)
  • Deterministic import/export
  • RFC‑compliant TOTP generator
  • 100% test coverage
  • CLI‑first, reproducible, auditable

Install

pip install desktop-2fa

Usage

# Add a new TOTP entry
desktop-2fa add

# Generate a code
desktop-2fa code <account_name>

# List entries
desktop-2fa list
