desktop‑2fa

A secure, offline, local‑first TOTP manager for developers, engineers, and privacy‑focused users.

Features

  • Encrypted vault (AES‑GCM + Argon2)
  • 100% test coverage
  • Deterministic import/export
  • RFC‑compliant TOTP generator
  • CLI‑first, no cloud, no telemetry
