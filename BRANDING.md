# Branding — desktop‑2fa

This project uses a minimal, technical visual identity inspired by the geometric
shield, hexagon and keyhole motif used in the hero illustration.

## Logo Variants
- `logo-outline.svg` — primary logo
- `logo-filled.svg` — alternative icon-style version
- `logo-mono.svg` — monochrome version for dark/light backgrounds
- `logo-horizontal.svg` — logo + wordmark

## Colors
- Primary: #58a6ff
- Secondary: #1f6feb
- Background: #0d1117
- Surface: #161b22
- Border: #21262d
- Text: #c5cdd8

## Usage Guidelines
- Keep the logo on dark backgrounds when possible.
- Do not stretch or distort the logo.
- Maintain clear space equal to half the logo height.
- Use the outline version as the default.
- Use the monochrome version when color is not available.
